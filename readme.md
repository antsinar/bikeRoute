# Bike Route

> Routing application for cyclist in Greece

### What is this?
An excuse for me to dive into the GIS world, starting with PostGIS and the GeoDjango framework. But also hopefully a step towards a solution to Athens' transit problems.  

### Core Dependencies
- [Django](https://www.djangoproject.com/) (web framework)
- [Django Rest Framework](https://www.django-rest-framework.org/) (rest api implementation)
- [Procrastinate](https://procrastinate.readthedocs.io/en/stable/) (task queue + worker)
- [Valhalla](https://github.com/valhalla/valhalla) (routing engine)
- [Martin](https://github.com/maplibre/martin) (vector tiles)
- [Geofabrik](https://www.geofabrik.de/) (greece's geospatial data)
- [Nominatim](https://nominatim.org/) (geocoder)
- [Postgresql](https://www.postgresql.org/) (main RDBMS)
- [PGAdmin](https://www.pgadmin.org/) (db administration)
- [Traefik](https://traefik.io/traefik) (reverse proxy for available services)
- [Docker](https://www.docker.com/) (container engine)
